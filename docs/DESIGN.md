# DESIGN.md

## Visual Theme & Atmosphere

- Tone: calm, trustworthy, professional.
- Product posture: ledger-first tool, not casino entertainment.
- Style direction: dark-first with restrained poker-felt influence.

## Color Palette

- Base dark: `#0F1512`
- Surface dark: `#16211B`
- Text light: `#E7F1EB`
- Accent green: `#2F6A3E`
- Optional secondary accent (muted): `#5A6F8A`

## Typography

- Keep typography highly legible and compact for data-heavy layouts.
- No playful display fonts for core data views.
- Use stronger contrast for key numeric values.

## Components

- Tables and cards should prioritize scannability.
- Inputs and action buttons must be touch-friendly on mobile.
- Settlement action requires strong confirmation cues.

## Do and Don't

- Do: clean spacing rhythm, clear grouping, explicit interaction feedback.
- Don't: neon, glowing effects, noisy backgrounds, red/green-only encoding.
