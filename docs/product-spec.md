# Product Spec

## Product Name

PokerChip Ledger

## One-line Positioning

A Texas Hold'em chip tracking and settlement system for offline games.

## Target Users

- Home-game players
- Session organizers
- Small club admins

## Core User Tasks

1. Record buy-ins, rebuys, and cash-outs quickly.
2. Track per-player chips and P/L in real time.
3. Generate final settlement with one action.

## Non-functional Goals

- High reliability for settlement math
- Fast interactions on mobile
- Accessible visual contrast (WCAG AA baseline)
